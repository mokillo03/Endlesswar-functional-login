Instalar node.js
librerias usadas:
dotenv
express 
bcryptjs
cookie-parser
jsonwebtoken


comandos del cmd:
cd (direccion)
node -v
npm init -y

npm i dotenv express bcryptjs cookie-parser jsonwebtoken

Para inicializar el programa:
npm run dev
